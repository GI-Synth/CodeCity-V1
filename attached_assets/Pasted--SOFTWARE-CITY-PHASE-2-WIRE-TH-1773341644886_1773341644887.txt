computeHealthScore(buildings: Building[]): { score: number, season: string }
Algorithm: testCoverageScore: avg coverage across all buildings (0–1) → buildings with fileType=test nearby get a 0.7 coverage bonus
to buildings in the same district cleanRatio: buildings with status=healthy / total buildings (0–1) complexityScore: 1 - (avgComplexity /
30) clamped to 0–1 testFileRatio: (count of test files / total files) clamped to 0–1
healthScore = (
 testCoverageScore * 0.40 +
 cleanRatio * 0.30 +
 complexityScore * 0.20 +
 testFileRatio * 0.10
)
season:
 >= 0.80 → "Summer"
 >= 0.60 → "Spring"
 >= 0.40 → "Autumn"
 default → "Winter"
return { score: Math.round(healthScore * 100), season }
Update GET /api/city/health to use this real computation instead of returning a hardcoded 55/Autumn.
